# MDCodeExtractor
Extracts all source code snippets and folder structure from all *.MD files

# How to use
In the command line, use the following command:
python md_code_extractor.py <MD_FILE_NAME.md> <Destination_folder>

Where:
- <MD_FILE_NAME.md>: the MD file name that contains the code snippets.
- <Destination_folder>: Where you want to extract them.

For example:
python md_code_extractor.py MD_FILE_NAME.md .

Where you want to extract all code from MD_FILE_NAME.md to the current folder.

# Requirements:
This code requires Python 3 to be installed on your PC.
